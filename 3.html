what is html

HTML stands for Hypertext Markup Language and is a
markup language used to create and structure web
pages. HTML uses tags and attributes to describe the
structure and formatting of a web page, including headings,
paragraphs, and links. HTML elements also tell the browser
how to display content, such as images, data tables, and
lists of bulleted points.
